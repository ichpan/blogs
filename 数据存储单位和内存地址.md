# 数据存储单位和内存地址

### 位(bit):

在计算机中数据存储的最小单位是一个二进制位Bit,一位可存储一个二进制数。

### 字节(Byte)

由于bit太小,无法用来表示数据的信息含义,所以把**8个连续的二进制**位组合在一起就构成一个字节(Byte)。 一般用字节作为计算机存储容量的基本单位。常用的单位有B、KB、MB、GB、TB

### 转换关系
~~~go
// 1个字节等于8位   1 byte = 8 bit

1KB = 2^10B = 1024B
1MB = 2^20B = 1024KB
1GB = 2^30B = 1024MB
1TB = 2^40B = 1024GB
~~~

### 内存地址和数据存放

~~~go
golang中的int为int64  64指的就是`位数`
int8: 1个字节
int16: 2个字节
int32: 4个字节
int64: 8个字节

float32: 4个字节
float64: 8个字节
~~~
